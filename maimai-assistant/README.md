# Maimai Assistant

> 🎯 脉脉招聘助手 - 嵌入式持久悬浮面板

脉脉招聘页面的 Chrome 扩展插件，帮助招聘者高效管理候选人。

## ✨ 核心功能

- 🤝 **批量添加好友** - 自动依次点击"加好友"
- 💬 **批量发消息** - 支持消息模板，自动发送招呼语
- 📋 **信息提取** - 批量/单个提取候选人详细信息
- 📤 **数据导出** - 支持 CSV/JSON 格式导出

## 🚀 安装方式

1. 下载本项目 `extension` 目录
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `extension` 目录

## 📖 使用说明

### 基本使用

1. 访问 [脉脉招聘页面](https://maimai.cn/ent/v41/recruit/talents?tab=1)
2. 页面左下角会出现持久悬浮面板
3. 勾选要操作的候选人
4. 点击相应操作按钮

### 批量加好友

1. 勾选候选人（可点击"全选"）
2. 点击「🤝 批量加好友」
3. 系统自动依次操作，间隔 5-10 秒

### 批量发消息

1. 勾选候选人
2. 在消息模板框输入内容（可用 `{name}` 替换姓名）
3. 点击「💬 批量发消息」

### 信息提取

1. 点击「📋 提取信息」
2. 自动提取当前页面所有候选人信息
3. 点击「导出 CSV」或「导出 JSON」保存

## ⚠️ 注意事项

- 批量操作间隔 5-10 秒，避免触发平台检测
- 请合理使用，遵守平台规则
- 数据仅保存在本地，不会上传
