# 提示词优化指南

## 概述

通过修改 [prompts.py](prompts.py) 中的提示词模板，你可以不断优化生成的小红书文案质量。

## 文件位置

```
backend/config/prompts.py
```

## 三种风格说明

### 版本 A：硬核技术风 (PROMPT_VERSION_A)

**目标受众**：技术人员、开发者、AI 从业者

**特点**：
- 专业术语多
- 数据驱动
- 技术原理分析
- 竞品对比

**优化方向**：
- 增加/减少技术深度
- 调整专业术语使用频率
- 改变数据呈现方式

**示例位置**：
```python
PROMPT_VERSION_A = """
你是一位专业的AI领域内容创作者...
"""
```

### 版本 B：轻松科普风 (PROMPT_VERSION_B)

**目标受众**：大众用户、AI 初学者、科技爱好者

**特点**：
- 口语化表达
- 生活化比喻
- 表情符号丰富
- 易于理解

**优化方向**：
- 调整表情符号使用频率
- 改变比喻方式
- 修改开头和结尾的情绪化表达

**示例位置**：
```python
PROMPT_VERSION_B = """
你是一位擅长科普的AI领域博主...
"""
```

### 版本 C：热点观点风 (PROMPT_VERSION_C)

**目标受众**：行业从业者、投资机构、科技媒体

**特点**：
- 有明确观点
- 分析行业影响
- 引发讨论
- 数据 + 观点结合

**优化方向**：
- 调整观点立场（激进/保守）
- 改变分析深度
- 修改互动问题设计

**示例位置**：
```python
PROMPT_VERSION_C = """
你是一位AI行业观察家...
"""
```

## 如何修改提示词

### 1. 修改提示词模板

直接编辑对应的 `PROMPT_VERSION_X` 变量：

```python
PROMPT_VERSION_A = """
你是一位专业的AI领域内容创作者，擅长将科技新闻转化为适合小红书的硬核技术文案。

请根据以下新闻内容，生成一篇小红书文案：

**要求：**
1. 标题：[在这里修改标题要求]
2. 内容：
   - [在这里修改内容要求]
   - [添加更多要求]
3. 风格：[修改风格描述]
4. 长度：[调整字数要求]
5. 标签：[修改标签要求]

**新闻内容：**
{title}
{summary}
{content}

**请直接输出生成的文案，格式如下：**
标题：[你的标题]
内容：[你的内容]
标签：[标签1] [标签2] [标签3]
"""
```

### 2. 修改占位符

提示词中有三个占位符会自动填充：

- `{title}` - 新闻标题
- `{summary}` - 新闻摘要
- `{content}` - 新闻完整内容

不要删除这些占位符，否则会导致错误。

### 3. 测试修改效果

修改后无需重启，重新点击"生成文案"按钮即可看到效果。

## AI 关键词过滤

### 修改 AI_KEYWORDS 列表

如果你想调整哪些新闻被认为是 AI 相关的，可以修改 `AI_KEYWORDS` 列表：

```python
AI_KEYWORDS = [
    # AI模型和框架
    'AI', 'GPT', 'ChatGPT', 'Claude', 'Llama',
    # 添加更多关键词...
]
```

### 添加新关键词

```python
AI_KEYWORDS = [
    'AI', 'GPT', 'ChatGPT',
    '你的关键词1', '你的关键词2',  # 添加新关键词
]
```

## 提示词优化技巧

### 1. 具体化要求

❌ 不好的示例：
```
生成吸引人的标题
```

✅ 好的示例：
```
标题：包含核心技术关键词，使用专业术语，字数控制在15-25字
```

### 2. 明确输出格式

❌ 不好的示例：
```
生成文案
```

✅ 好的示例：
```
**请直接输出生成的文案，格式如下：**
标题：[你的标题]
内容：[你的内容]
标签：[标签1] [标签2] [标签3]
```

### 3. 使用示例

在提示词中添加示例可以提高生成质量：

```python
PROMPT_VERSION_A = """
你是一位专业的AI领域内容创作者。

**示例：**
标题：GPT-5 发布：性能提升 300%，推理成本降低 50%
内容：OpenAI 今日正式发布 GPT-5...
标签：#AI #GPT5 #黑科技

现在请根据以下新闻内容生成文案：
...
"""
```

### 4. 调整温度参数

如果生成结果过于随机或过于死板，可以调整 AI 服务的温度参数（在 [services/ai_service.py](../services/ai_service.py) 中）：

```python
# 在 _generate_with_openai 方法中
temperature=0.8  # 范围 0.0-2.0，越高越随机
```

## 常见问题

### Q1: 修改后为什么没有效果？

**A**: 确保：
1. 保存了文件
2. 没有语法错误（引号、缩进）
3. 重新点击生成按钮

### Q2: 如何增加更多风格版本？

**A**:
1. 在 `prompts.py` 中添加新的 `PROMPT_VERSION_D`
2. 在 `PROMPTS` 字典中添加 `'D': PROMPT_VERSION_D`
3. 修改前端请求中的 `versions` 参数

### Q3: 生成速度太慢怎么办？

**A**:
1. 减少生成的版本数量（只生成 A 和 B）
2. 减少 `{content}` 的长度限制
3. 使用更快的模型（如 GPT-3.5-turbo）

### Q4: 如何提高生成质量？

**A**:
1. 优化提示词（更具体、更明确的要求）
2. 在提示词中添加示例
3. 调整温度参数
4. 尝试不同的模型

## 版本控制

建议对提示词进行版本控制：

```bash
# 备份当前版本
cp prompts.py prompts.py.backup.$(date +%Y%m%d)

# 测试新版本
# 修改 prompts.py

# 如果效果好，保留
# 如果不好，恢复备份
cp prompts.py.backup.20250113 prompts.py
```
