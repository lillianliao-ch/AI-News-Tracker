# AI News Tracker 使用指南

## 🎯 功能说明

这是一个 AI 资讯追踪与小红书文案生成系统，可以帮助你：
1. 自动聚合多个科技媒体的 AI 相关资讯
2. AI 驱动自动分类、摘要
3. 一键生成 3 种风格的小红书文案，方便复制使用

---

## 📖 使用流程

### 1. 浏览资讯

访问：http://localhost:4321

**功能：**
- 查看最新的 AI 行业资讯
- 按分类筛选（产品、模型、融资、观点）
- 点击标题跳转到原文链接
- 点击"刷新资讯"获取最新数据

### 2. 生成小红书文案

有两种方式生成文案：

#### 方式一：在资讯列表页直接生成

1. 在资讯卡片上点击 **"✍️ 生成文案"** 按钮
2. 等待几秒，系统会自动生成 3 个版本的文案
3. 点击 **"📋 复制"** 按钮复制文案
4. 粘贴到小红书发布

#### 方式二：在专门的生成页面生成

1. 点击左侧菜单的 **"✍️ 内容生成"**
2. 浏览资讯列表
3. 点击资讯下的 **"🚀 一键生成全部版本"** 按钮
4. 查看生成的 3 个版本
5. 点击 **"📋 复制全部"** 按钮
6. 粘贴到小红书发布

### 3. 三种文案风格说明

| 版本 | 风格 | 适用场景 |
|------|------|----------|
| **A - 硬核技术风** | 专业但不晦涩，用数据说话 | 技术人员、开发者、AI 从业者 |
| **B - 轻松科普风** | 口语化，多用表情符号 😱🔥✨ | 大众用户、AI 初学者、科普内容 |
| **C - 热点观点风** | 有观点有立场，引发讨论 | 行业观察者、关注趋势者 |

## 💡 使用技巧

### 技巧 1：选择合适的版本
- 如果你的粉丝是技术人员，用版本 A
- 如果是普通大众，用版本 B
- 如果想引发讨论，用版本 C

### 技巧 2：修改文案
生成的文案是基础版本，你可以根据需要：
- 添加个人观点
- 调整语气和表达
- 补充更多细节
- 添加相关话题标签

### 技巧 3：发布时机
- 工作日的中午 12-13 点
- 晚上的 20-22 点
- 周末的任何时间

## ⚙️ 配置 API Keys

目前系统使用的是占位符 API Keys，需要配置才能使用：

1. 编辑 `backend/.env` 文件
2. 替换以下内容：
```bash
OPENAI_API_KEY=sk-你的OpenAI密钥
ANTHROPIC_API_KEY=sk-ant-你的Anthropic密钥
```

3. 重启后端服务

## 🚀 快捷操作

### 常用页面
- 资讯列表：http://localhost:4321
- 内容生成：http://localhost:4321/generate
- 已发布（占位）：http://localhost:4321/published

- 点击页面右上角的 **"🔄 刷新资讯"** 按钮

- 左侧边栏显示实时统计信息
- 或者访问：`curl http://localhost:8000/api/stats`

## ❓ 常见问题

### Q: 生成的文案质量如何？
A: 文案质量取决于 AI 模型，建议使用 GPT-4 或 Claude 3.5 Sonnet 以获得最佳效果。

### Q: 可以修改生成的文案吗？
A: 当然可以！生成的文案只是基础版本，你可以随意修改、添加个人特色。

### Q: 一次生成多长时间？
A: 通常 3-10 秒，取决于 API 响应速度。

### Q: 支持其他平台吗？
A: 目前主要针对小红书优化，但文案也适用于微博、朋友圈等平台。

### Q: 如何添加更多资讯源？
A: 编辑 `backend/config/base_config.py` 中的 `SOURCES_CONFIG` 配置。

## 🎓 最佳实践

1. **先浏览，后生成** - 先快速浏览资讯，选择有价值的再生成文案
2. **及时发布** - AI 资讯时效性强，建议生成后尽快发布
3. **保持原创** - 在生成文案的基础上添加自己的观点和分析
4. **关注反馈** - 根据粉丝反馈调整内容风格
