## 🚀 使用指南

### 快速开始

```bash
# 1. 启动后端
cd backend
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload

# 2. 启动前端
cd frontend
npm run dev

# 3. 访问应用
open http://localhost:4321

# 4. 生成文案
- 浏览新闻
- 点击"✍️ 生成文案"
- 等待10-30秒
- 查看三个版本
- 点击"📋 复制"
```

### 爬取新闻

```bash
cd backend
python -m tasks.crawler

# 查看日志
tail -f /tmp/backend.log
```

### 优化提示词

```bash
# 编辑提示词
vim backend/config/prompts.py

# 重启后端（应用更改）
# uvicorn会自动重载
```
