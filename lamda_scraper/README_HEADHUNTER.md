## 🔥 实战技巧

### 技巧1: 优先联系组合

**最高转化率:**
```
Tier A + Hot + 公司邮箱 + NeurIPS/ICML ≥ 3篇
→ 转化率 30-40%
```

**示例候选人:**
```
姜伟 | Tier A | Hot | 南京理工
- ICML×4 + NeurIPS×5 + ICLR×1 + TPAMI×2
- 2025.10入职 (刚入职，可能不稳定)
- jiangw@lamda.nju.edu.cn

策略:
1. 2-3个月后联系 (过了适应期)
2. 强调学术+工业结合的机会
3. 可以提供团队管理位置
```

### 技巧2: 个性化消息要素

**必须包含 (回复率+30%):**
- ✅ 提及具体论文
- ✅ 提及导师
- ✅ 说明为什么匹配
- ✅ 给出明确下一步

**示例:**
```
您好 姜伟，

我是[公司]的猎头[名字]，关注到您在ICML 2023发表的
"Neural Architecture Search with Evolutionary Computation"，
非常认同您在演化计算与NAS结合的创新工作。

了解到您目前在南京理工任教，我们正在为DeepMind寻找
强化学习研究员，相信这与您的研究高度契合。

您的导师张利军教授在演化计算领域的贡献我也非常敬佩。

是否方便本周进行10分钟简短交流？

祝好，
[你的名字]
```

### 技巧3: 多渠道触达

**优先级:**
1. **公司邮箱** (15分) - 在职 = 有意向
2. **LinkedIn消息** (8分) - 专业度高
3. **GitHub个人邮箱** (3分) - 备选

**组合策略:**
```
Day 1: 公司邮箱 (个性化消息)
Day 3: LinkedIn (连接请求)
Day 7: 跟进消息 (如果未回复)
```
