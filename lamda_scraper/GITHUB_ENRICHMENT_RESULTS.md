## 🔧 技术说明

### 评分体系 (100分制)

**总分 = 活跃度 (40) + 影响力 (30) + 质量 (30)**

#### 1. 活跃度评分 (0-40分)
- 公开仓库数: 2分/仓库，最多20分
- 账号年限: 3分/年，最多10分
- 最近更新: 30天内=10分，90天内=5分

#### 2. 影响力评分 (0-30分)
- Followers: Followers/10，最多15分
- Stars: Stars/5，最多10分
- Forks: Forks/10，最多5分

#### 3. 质量评分 (0-30分)
- 有 Bio: 5分
- 有公司信息: 5分
- Hireable: 10分
- 有邮箱: 5分
- 仓库质量 (平均Stars≥10): 5分

### 技术栈自动识别

从以下维度检测:
1. Bio 关键词
2. 仓库名和描述
3. 编程语言统计

支持技术栈:
- Python, JavaScript, Web
- Machine Learning, Deep Learning
- Computer Vision, NLP
- Reinforcement Learning
- Data Science
