# PDF Skill 测试指南

## 📋 测试清单

### 方式 A：在 Claude Code 中交互式测试

**Step 1：安装 PDF Skill（如果还没安装）**
```bash
/plugin marketplace add anthropics/pdf
```

**Step 2：对 Claude 说以下测试指令**

#### 测试 1：基础文本提取
```
使用 pdf skill 解析这个 PDF 文件：
/Users/lillianliao/Desktop/CV_ZhanzhaoLIANG_20251016.pdf

请提取：
1. 基本信息（姓名、电话、邮箱）
2. 工作经历（公司、职位、时间）
3. 教育背景
4. 技能标签

输出为 JSON 格式。
```

#### 测试 2：表格提取
```
使用 pdf skill 解析同一个 PDF，
特别关注：
- 所有表格数据
- 表格的列名和行数据
- 将表格输出为结构化的 JSON 数组
```

#### 测试 3：元数据提取
```
使用 pdf skill 提取这个 PDF 的元数据：
- 作者
- 创建时间
- 修改时间
- PDF 版本
- 总页数
```

#### 测试 4：复杂布局处理
```
使用 pdf skill 分析这个简历的布局：
- 是否有表格？
- 是否有多栏布局？
- 是否有图片？
- 如果有，请描述图片的位置和内容
```

### 方式 B：在你的 Streamlit 应用中测试

**测试步骤：**

1. **启动应用**

2. **进入"添加人才"页面**

   - 上传：`CV_ZhanzhaoLIANG_20251016.pdf`

4. **观察解析结果**
   - 查看 AI 提取的结构化信息
   - 检查是否有格式错误
   - 验证关键字段（姓名、公司、职位）是否正确

## 🎯 成功标准

### PyPDF2 预期问题
- ❌ 表格数据可能混乱
- ❌ 多栏布局可能错位
- ❌ 特殊字符可能丢失
- ❌ 需要手动清洗数据

- ✅ 表格数据结构化
- ✅ 布局正确识别
- ✅ 保留元数据
- ✅ 无需清洗代码
