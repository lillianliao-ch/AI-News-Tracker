# Boss 职位自动发布助手 - 使用指南

## 快速开始

### 1. 安装扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `ai-job-publisher/extension` 文件夹
6. 确认扩展已加载成功

### 2. 准备 Excel 文件

确保 Excel 包含以下必填列：

| 列名 | 说明 | 示例 |
|-----|------|-----|
| 职位名称 | 职位标题 | 高德-高级算法工程师-北京 |
| 职位类型（子类） | Boss 职位分类 | 大模型算法 |
| 工作城市 | 城市（可多个） | 中国-北京-北京,中国-浙江-杭州 |
| 学历 | 学历要求 | 硕士 |
| 工作年限 | 经验要求 | 八年以上 |
| 月薪下限 | 最低月薪（元） | 70000 |
| 月薪上限 | 最高月薪（元） | 110000 |
| 职位描述 | JD 描述 | 负责搜索算法... |
| 岗位要求 | 任职要求 | 熟悉机器学习... |
| Boss关键词 | 职位标签 | ['LLM', 'NLP', '深度学习'] |

### 3. 发布职位

1. 登录 Boss 直聘
2. 访问职位发布页面：`https://www.zhipin.com/web/chat/job/edit`
3. 点击扩展图标，打开面板
4. 点击"上传 Excel 文件"，选择准备好的 Excel
5. 预览职位列表，确认无误
6. 点击"开始自动发布"
7. 等待自动发布完成（不要关闭页面）
8. 下载发布报告（CSV 格式）

## 注意事项

### 发布速度
- 每个职位发布间隔 20-30 秒（随机）
- 27 个职位预计需要 9-13 分钟
- 不要手动刷新页面或关闭浏览器

### 字段匹配
- **工作年限映射**：
  - 八年以上 → 8年以上
  - 五年以上 → 5-10年
  - 三年以上 → 3-5年
  - 一年以上 → 1-3年
  - 应届生 → 1年以内

- **学历映射**：直接使用（本科、硕士、博士等）

- **职位类型**：
  - 大模型算法 → 互联网/AI > 人工智能 > 大模型算法
  - 图像算法 → 互联网/AI > 人工智能 > 图像算法
  - 数据挖掘 → 互联网/AI > 数据 > 数据挖掘
  - 其他技术职位 → 互联网/AI > 其他技术职位

- **工作城市**：
  - 自动提取第一个城市（如"中国-北京-北京,中国-浙江-杭州" → "北京"）
  - 自动点击"使用该地址"确认

### 错误处理
- 如果某个职位必填项缺失，会跳过并记录到报告
- 如果 Boss 页面结构变化，可能导致填写失败
- 建议先用 1-2 个职位测试，确认无误后再批量发布

## 常见问题

**Q: 为什么有些职位类型选择失败？**  
A: 可能是 Excel 中的"职位类型（子类）"值不在映射表中。请检查 `content.js` 的 `CONFIG.MAPPINGS.jobType`，确保包含你的职位类型。

**Q: 工作地点填写失败怎么办？**  
A: 可能是城市名称格式不对。确保 Excel 中的"工作城市"格式为"中国-省份-城市"。

**Q: Boss关键词无法添加？**  
A: 确保 Excel 中的"Boss关键词"列格式为 `['关键词1', '关键词2']`（Python list 格式的字符串）。

**Q: 可以同时发布多个 Excel 吗？**  
A: 不可以，请分批上传和发布。
