# Boss 职位自动发布助手

## 功能

根据 Excel 表格自动在 Boss 直聘批量发布职位

## 使用步骤

1. **加载扩展**
   - 打开 Chrome，访问 `chrome://extensions`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 `ai-job-publisher/extension` 文件夹

2. **准备 Excel**
   - 确保 Excel 包含以下列：
     - 职位名称
     - 职位类型（子类）
     - 工作城市
     - 学历
     - 工作年限
     - 月薪下限、月薪上限
     - 职位描述
     - 岗位要求
     - Boss关键词

3. **开始发布**
   - 打开 Boss 直聘职位发布页面：`https://www.zhipin.com/web/chat/job/edit`
   - 点击扩展图标
   - 上传 Excel 文件
   - 点击"开始自动发布"
   - 等待自动发布完成

## 字段映射

| Excel 字段 | Boss 字段 | 说明 |
|-----------|----------|------|
| 职位名称 | 职位名称 | 直接填写 |
| 职位类型（子类） | 职位类型 | 自动匹配（大模型算法、图像算法等） |
| 工作年限 | 经验 | 八年以上→8年以上 |
| 学历 | 学历 | 硕士→硕士 |
| 工作城市 | 工作地点 | 自动提取并确认 |
| 月薪下限、月薪上限 | 薪资范围 | 自动转换为万元 |
| 职位描述 + 岗位要求 | 职位描述 | 自动合并 |
| Boss关键词 | 职位关键词 | 逐个添加（最多8个） |

## 注意事项

- 每个职位发布间隔 20-30 秒（防止触发平台风控）
- 发布完成后会自动导出发布报告（CSV 格式）
- 如果某个职位发布失败，会跳过并继续下一个
- 建议分批发布，每次不超过 10 个职位

## 故障排查

- **扩展无法加载**：检查 manifest.json 格式
- **页面检测失败**：确保在 `https://www.zhipin.com/web/chat/job/edit` 页面
- **字段填写失败**：打开 Console 查看详细日志
