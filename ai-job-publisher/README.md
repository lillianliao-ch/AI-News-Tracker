# Boss 职位自动发布助手

## 项目简介

这是一个 Chrome 扩展，可以根据 Excel 表格自动在 Boss 直聘批量发布职位，节省 HR 大量重复劳动。

## 主要功能

- ✅ Excel 表格解析（支持 .xlsx 格式）
- ✅ 自动填写职位基本信息
- ✅ 智能匹配职位类型（多级分类）
- ✅ 自动处理工作地点选择弹窗
- ✅ 批量添加职位关键词
- ✅ 防风控间隔发布（20-30秒/职位）
- ✅ 导出发布报告（成功/失败记录）

## 技术栈

- Chrome Extension (Manifest V3)
- SheetJS (Excel 解析)
- 原生 JavaScript (DOM 操作)

## 字段映射规则

### 工作年限
- 八年以上 → 8年以上
- 五年以上 → 5-10年
- 三年以上 → 3-5年
- 一年以上 → 1-3年
- 应届生 → 1年以内

### 职位类型
- 大模型算法 → 互联网/AI > 人工智能 > 大模型算法
- 图像算法 → 互联网/AI > 人工智能 > 图像算法
- 算法工程师 → 互联网/AI > 人工智能 > 算法工程师
- 数据挖掘 → 互联网/AI > 数据 > 数据挖掘
- 其他技术职位 → 互联网/AI > 其他技术职位

### 薪资
- 自动从"元"转换为"万元"
- 例：70000 → 7万，110000 → 11万

## 使用建议

1. **首次使用**：先用 1-2 个职位测试，确认字段映射正确
2. **分批发布**：建议每次不超过 10 个职位，避免触发风控
3. **监控日志**：打开 Console（F12）查看详细执行日志
4. **保存进度**：每批发布完会自动导出报告，可用于跟踪进度

## 注意事项

⚠️ **重要提醒**：
- 此工具仅用于合法合规的职位发布
- 请确保 Excel 中的职位信息真实准确
- 批量发布时请遵守 Boss 直聘的使用规范
- 不建议在高峰时段大量发布（易触发风控）

## 故障排查

### 扩展无法加载
- 检查 Chrome 版本（需要 Manifest V3 支持）
- 确认开发者模式已开启
- 查看扩展错误日志

### 页面检测失败
- 确保在职位发布页面：`https://www.zhipin.com/web/chat/job/edit`
- 刷新页面后重试
- 检查 Boss 是否需要重新登录

### 字段填写失败
- 打开 Console 查看详细错误
- 检查 Boss 页面结构是否变化
- 对比 Excel 字段值与 Boss 选项是否匹配
