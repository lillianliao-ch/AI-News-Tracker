## 📞 故障排除

### 无法访问网站
1. **检查服务器状态**
   ```bash
   curl -I http://localhost:8080
   ```

2. **检查网络连接**
   ```bash
   ping 192.168.3.97
   ```

3. **检查端口是否开放**
   ```bash
   lsof -i :8080
   ```

### 常见问题
- **连接被拒绝**: 检查防火墙设置
- **页面无法加载**: 确认服务器正在运行
- **移动端显示异常**: 清除浏览器缓存
