# 🚀 猎头联盟网站 - 快速开始

## 📋 系统要求
- Python 3.7+
- pip3

## ⚡ 快速启动

### 方法1：使用启动脚本（推荐）
```bash
./start_server.sh
```

### 方法2：手动启动
```bash
# 1. 安装依赖
pip3 install -r requirements.txt

# 2. 启动应用
python3 app.py
```

## 🌐 访问网站
打开浏览器访问：**http://localhost:8080**

## 🎯 功能演示

### 1. 企业注册
- 访问首页，点击"企业注册"
- 填写企业信息并创建账户

### 2. 发布职位
- 登录后进入管理面板
- 点击"发布新职位"
- 填写完整的职位信息

### 3. 搜索职位
- 在首页或职位搜索页面
- 使用多维度筛选功能
- 按企业、职位、地点等条件搜索

### 4. 查看职位详情
- 点击任意职位卡片
- 查看完整的职位信息
- 获取联系方式

## 📱 主要页面

| 页面 | 功能 | 访问地址 |
|------|------|----------|
| 首页 | 平台介绍、快速搜索 | `/` |
| 职位搜索 | 多维度筛选、分页浏览 | `/jobs` |
| 企业注册 | 创建企业账户 | `/register` |
| 企业登录 | 登录管理面板 | `/login` |
| 管理面板 | 职位管理、统计信息 | `/dashboard` |
| 发布职位 | 创建新职位 | `/post-job` |

## 🔧 常见问题

### Q: 端口5000被占用怎么办？
A: 应用已配置使用端口5001，如果仍有问题，可以修改 `app.py` 中的端口号。

### Q: 如何重置数据库？
A: 删除 `headhunter_alliance.db` 文件，重新启动应用。

### Q: 如何添加测试数据？
A: 注册企业账户后，通过管理面板发布职位即可。
