## 一、项目概述

猎头协作平台是一个基于代理模式的B2B协作平台，专门为猎头公司、猎头顾问和SOHO顾问提供职位发布、候选人推荐、协作管理和利益分配的一站式解决方案。

### 核心特点 ✅
- 代理模式：候选人和客户企业通过猎头顾问代理，无需直接登录平台
- 协作导向：支持多个猎头之间的职位协作和候选人推荐
- 利益分配：基于职位级别的灵活分成机制
- **⭐ AI智能化**: 智能文档识别、简历匹配、批量导入等AI功能

## 二、用户角色体系 ✅ **已完成**

### 2.1 平台管理员 (Platform Admin) ✅
- **职责**：审核猎头公司管理员和SOHO顾问的注册申请，管理平台整体运营
- **权限**：
  - 管理平台整体数据（职位、简历、协作记录）
  - 处理违规操作（拉黑、封禁）
  - 维护系统配置（通知策略、权限规则等）
  - 审核候选人维护人变更申请 ❌ **未实现**
- **审核范围**：
  - ✅ **公司管理员注册审核**：验证公司资质和营业执照，审核猎头公司注册申请
  - ✅ **SOHO顾问注册审核**：验证个人资质和从业背景，审核独立猎头申请
  - ❌ **不负责咨询顾问审核**（由公司管理员负责）
- **前端界面权限** ✅：
  - User Management: 显示待审核的公司管理员和SOHO列表
  - Company Management: 公司审核和管理，查看所有公司信息
  - 不显示咨询顾问审核功能
- **公司信息访问权限** ✅：
  - 可查看和管理所有平台公司信息
  - 可审核公司注册申请并批准/拒绝
  - 拥有最高级别的公司数据访问权限

### 2.2 公司管理员 (Company Admin) ✅
- **注册流程** ✅：输入企业名称、营业执照、联系人信息（手机号/邮箱）
- **审核机制** ✅：平台管理员审核通过后，管理员账号和公司同时激活
- **管理功能** ✅：可创建子账号/员工账号，查看与本公司相关的职位、简历、协作数据
- **审核权限** ✅：
  - ✅ **咨询顾问审核**：审核并邀请顾问加入本公司，审核通过后自动绑定到本公司
  - ✅ **团队管理**：管理公司内部咨询顾问团队
  - ❌ **不负责其他角色审核**（平台管理员和SOHO由平台管理员审核）
- **前端界面权限** ✅：
  - Consultant Approval: 专门的咨询顾问审核页面
  - My Company: 本公司信息查看和编辑管理
  - 不显示其他角色审核功能
- **公司信息访问权限** ✅：
  - 可查看和编辑本公司完整信息（基本信息、联系方式、地址等）
  - 可查看本公司统计数据（团队成员、职位、投递、候选人等）
  - 可管理公司团队成员信息
  - 具有公司信息的完整编辑权限

### 2.3 咨询顾问 (Consultant) - 公司员工 ✅
- **注册流程** ✅：通过手机号/邮箱注册，填写个人信息，无需指定公司
- **审核机制** ✅：由公司管理员在"Consultant Approval"页面审核，平台管理员不参与
- **绑定流程** ✅：审核通过后自动绑定到审核公司，获得公司内部权限
- **核心功能** ✅：
  - 发布职位、协作职位
  - 投递候选人简历并跟进进展
  - 代理客户企业和候选人进行平台操作
  - 使用公司资源和团队协作功能
- **前端界面权限** ✅：
  - 无用户审核功能
  - 专注于业务操作：职位管理、候选人管理、协作网络
  - Company Info: 查看本公司信息（只读）
  - 丰富的数据分析和报表功能
- **公司信息访问权限** ✅：
  - 可查看本公司基本信息（公司名称、行业、规模、联系方式等）
  - 可查看本公司统计数据和团队成员列表
  - 只读权限，无法编辑任何公司信息
  - 明确告知权限边界，如需修改需联系公司管理员

### 2.4 SOHO 顾问 (Individual Headhunter) ✅
- **注册流程** ✅：填写个人信息（姓名、联系方式、擅长领域），选择"SOHO"角色
- **审核机制** ✅：平台管理员人工审核确保资质合规，无公司管理员参与
- **功能权限** ✅：与咨询顾问类似的业务功能，但无公司绑定
- **独立运营** ✅：不受公司限制，可与任何公司或顾问协作
- **前端界面权限** ✅：
  - 无用户审核功能
  - 专注于业务操作：职位管理、候选人管理、协作网络
  - 独立的个人业务分析和统计
  - 无公司相关功能界面
- **公司信息访问权限** ✅：
  - 无公司归属，不能访问任何公司信息
  - 独立运营，无需公司信息支持
  - 可通过协作网络与其他公司合作

### 2.5 代理角色说明 ✅
- **候选人**：由猎头顾问代理操作，不登录平台
- **客户企业**：由猎头顾问代理发布需求，不登录平台

## 三、核心功能模块

### 3.1 账号与权限管理 ✅ **已完成**

#### 3.1.1 用户注册流程 ✅

**流程1: 公司管理员注册** ✅
```
填写公司信息 → 提交申请 → 平台管理员审核 → 公司和账号同时激活
```
- 必填信息：个人信息 + 公司名称 + 营业执照号 ✅
- 审核内容：公司资质验证、营业执照真实性 ✅
- 审核周期：3-5个工作日 ✅
- 激活后：获得公司管理权限，可邀请咨询顾问 ✅

**流程2: SOHO顾问注册** ✅
```
填写个人信息 → 提交申请 → 平台管理员审核 → 账号激活
```
- 必填信息：用户名、邮箱、手机、密码 ✅
- 审核周期：1-3个工作日 ✅
- 激活后：可立即开始使用平台功能 ✅

**流程3: 咨询顾问注册 (公司申请模式)** ✅
```
填写个人信息 → 选择目标猎头公司 → 提交申请 → 公司管理员审核 → 绑定公司 → 账号激活
```
- **注册信息** ✅：
  - 必填信息：用户名、邮箱、手机、密码
  - 公司选择：从已审核通过的猎头公司中选择要申请加入的公司
- **申请流程** ✅：
  - 系统显示所有已审核通过的猎头公司供选择
  - 顾问选择目标公司后提交申请，系统自动创建用户账号和申请记录
  - 对应公司管理员在"Consultant Applications"页面查看和处理申请
- **审核功能** ✅：
  - 申请列表：显示申请人信息（用户名、邮箱、手机）和申请时间
  - 筛选功能：按状态筛选（全部/待审核/已通过/已拒绝）
  - 审核操作：通过/拒绝，拒绝时可填写拒绝理由
  - 状态追踪：记录审核人、审核时间、审核结果
- **自动绑定** ✅：
  - 审核通过：顾问自动绑定到审核公司，账号状态变为活跃
  - 审核拒绝：账号保持待审核状态，可申请其他公司
- **审核周期**：1-7个工作日 ✅
- **激活后**：自动绑定到审核公司，获得公司内部权限 ✅

#### 3.1.2 权限控制 ✅ **已完成**

**核心权限原则** ✅
- 基于角色的权限管理体系 (RBAC)
- 公司级数据隔离和共享控制
- 操作日志和审计跟踪

**角色权限边界** ✅
```
权限边界规则:
├── 平台管理员: 可审核公司管理员和SOHO，不可审核咨询顾问
├── 公司管理员: 可审核咨询顾问，不可审核其他角色
├── 咨询顾问: 无审核权限，仅业务操作
└── SOHO顾问: 无审核权限，仅业务操作

数据访问控制:
├── 平台管理员: 全平台数据访问权限
├── 公司管理员: 本公司数据 + 协作数据访问权限
├── 咨询顾问: 本公司数据 + 个人业务数据访问权限
└── SOHO顾问: 个人业务数据 + 协作数据访问权限
```

**API端点权限验证** ✅
- 所有审核相关API都有严格的角色验证
- 平台管理员无法调用咨询顾问审核API
- 公司管理员无法调用其他角色审核API
- 咨询顾问和SOHO无法访问任何审核API
