About

LinkedIn-Atlas Candidate Analyzer v2.0.0
Analyze LinkedIn candidates using ChatGPT Atlas AI with automated data extraction and export.

How it works:
1. Visit a LinkedIn profile page
2. Click the "Analyze with AI" button
3. The extension sends the profile to ChatGPT Atlas
4. AI analysis is automatically saved to database and Excel
