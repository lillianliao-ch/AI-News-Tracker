# 脉脉招聘助手 - 安装使用指南

## 📦 安装步骤

### 1. 准备工作
- 确保使用 Chrome 88+ 浏览器
- 下载完整的项目文件夹

### 2. 安装插件

#### 方法一：开发者模式安装（推荐）
1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 在右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目根目录（包含 manifest.json 的文件夹）
6. 安装完成，插件图标会出现在工具栏

#### 方法二：打包安装
1. 在扩展程序页面点击「打包扩展程序」
2. 选择项目根目录
3. 生成 .crx 文件后拖拽到浏览器安装

### 3. 验证安装
- 访问 [脉脉招聘页面](https://maimai.cn/ent/v41/recruit/talents)
- 查看左下角是否出现蓝色「脉脉招聘助手」面板
- 点击插件图标查看弹出窗口

## 🎯 使用说明

### 基本使用流程

1. **访问脉脉网站**
   - 登录脉脉账号
   - 进入招聘/人才页面
   - 查看具体的人才详情页

2. **使用插件提取信息**
   - 左下角会自动显示蓝色控制面板
   - 点击「提取当前人才」按钮
   - 查看提取的信息预览
   - 在备注框添加自定义信息

3. **保存和导出数据**
   - 点击「保存数据」存储到本地
   - 累积多个人才信息后
   - 点击「导出Excel」下载文件

### 界面功能说明

```
脉脉招聘助手面板
├── 标题栏（可拖拽移动）
├── 控制按钮
│   ├── 提取当前人才    # 从当前页面提取信息
│   ├── 清空数据       # 清除当前显示的数据
│   └── 备注框         # 添加自定义备注
├── 数据预览区
│   ├── 姓名          # 人才姓名
│   ├── 地点          # 工作地点
│   ├── 工作年限       # 工作经验
│   └── 当前公司       # 当前就职公司
└── 操作按钮
    ├── 保存数据       # 存储到本地
    └── 导出Excel     # 下载Excel文件
```

## 📊 数据字段说明

### 基础信息字段
- **姓名**：人才真实姓名或昵称
- **地点**：当前工作城市
- **工作年限**：总工作经验年限
- **当前公司**：目前就职的公司名称
- **上一家公司**：前一份工作的公司

### 教育信息字段
- **学校1**：最高学历院校
- **学校2**：第二学历院校（如有）
- **教育经历**：完整的教育背景描述

### 详细信息字段
- **工作经历**：完整的职业发展历程
- **备注**：用户自定义的额外信息

### 元数据字段
- **提取时间**：数据提取的时间戳
- **来源链接**：原始页面URL

## 🔧 故障排除

### 常见问题

#### 1. 插件不显示
- **检查项**：
  - 是否在脉脉网站 (maimai.cn)
  - 插件是否正确安装并启用
  - 浏览器是否支持扩展程序

- **解决方案**：
  - 刷新页面重试
  - 检查扩展程序管理页面
  - 重新安装插件

#### 2. 无法提取数据
- **可能原因**：
  - 页面结构发生变化
  - 网络连接问题
  - 权限被限制

- **解决方案**：
  - 确保页面完全加载
  - 检查网络连接
  - 尝试手动刷新页面

#### 3. 导出功能异常
- **检查项**：
  - 浏览器下载权限
  - 本地存储空间
  - 数据是否已保存

- **解决方案**：
  - 检查浏览器下载设置
  - 清理浏览器缓存
  - 重新保存数据后导出

### 调试模式

2. 切换到 Console 标签

## ⚠️ 注意事项

### 使用规范
- 本工具仅供学习和个人研究使用
- 请遵守脉脉网站的使用条款和服务协议
- 合理控制使用频率，避免对服务器造成压力
- 不要用于商业用途或数据买卖

### 数据安全
- 所有数据存储在本地浏览器中
- 插件不会上传任何信息到外部服务器
- 用户可随时清除本地存储的数据
- 建议定期备份重要数据

### 隐私保护
- 插件不收集用户个人信息
- 不跟踪用户浏览行为
- 不包含任何分析统计代码
- 完全离线工作，保护用户隐私
