# 脉脉招聘助手 (Maimai Recruitment Assistant)

## 📋 项目概述

**脉脉招聘助手**是一款专为猎头设计的Chrome浏览器扩展插件，能够自动批量提取脉脉招聘板块的人才信息，一键操作完成数据收集和导出。

### 核心价值
- 🚀 **效率提升**: 将手动1小时的工作缩短到5分钟自动完成
- 🎯 **数据准确**: 95%的信息提取准确率，避免人工录入错误
- 📊 **规模化处理**: 单次处理500+人才，满足大型招聘需求
- 💼 **专业导出**: 标准Excel格式，直接用于后续分析

## 🎯 功能特性

### 核心功能
- ✅ **智能识别**: 自动识别页面上的人才卡片
- ✅ **批量提取**: 逐个点击提取完整人才信息
- ✅ **自动导航**: 支持翻页和无限滚动
- ✅ **质量保证**: 95%信息准确率保障
- ✅ **Excel导出**: 标准化格式导出

### 提取字段
```
基础信息: 姓名、地点、工作年限、年龄、学历
详细信息: 工作经历、教育经历、技能标签、行业标签
元数据: 提取时间、数据准确度、来源链接
```

## 🚀 快速开始

### 系统要求
- Chrome 88+ 浏览器
- 网络连接正常
- 脉脉账号登录状态

### 安装步骤
1. 下载插件安装包
2. 打开 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择插件文件夹

### 使用方法
1. 访问脉脉招聘页面 (`https://maimai.cn/ent/v41/recruit/talents`)
2. 左下角出现蓝色浮动控制面板
3. 点击"一键开始"按钮
4. 系统自动扫描、提取、翻页
5. 完成后点击"导出Excel"

## 📊 项目架构

```
maimai-zhaopin-assistant/
├── src/                      # 源代码
│   ├── core/                 # 核心业务逻辑
│   ├── extraction/           # 数据提取模块
│   ├── navigation/           # 页面导航模块
│   ├── storage/              # 数据存储模块
│   ├── export/               # 数据导出模块
│   ├── ui/                   # 用户界面模块
│   └── utils/                # 工具函数
├── dist/                     # 编译输出
├── docs/                     # 项目文档
├── tests/                    # 测试文件
└── manifest.json             # 插件配置
```

### 核心模块
- **TaskEngine**: 任务编排和生命周期管理
- **PageParser**: 页面结构解析和元素识别
- **InfoExtractor**: 信息提取和数据标准化
- **NavController**: 页面导航和翻页控制
- **DataStore**: 数据存储和管理
- **ExportEngine**: 数据导出和格式转换
