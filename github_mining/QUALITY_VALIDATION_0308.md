### 数据验证方法

```sql
-- 查询高质量候选人
SELECT
    name,
    github_url,
    website_quality_score,
    CASE
        WHEN extracted_work_history IS NOT NULL
             AND extracted_work_history != ''
             AND extracted_work_history != '[]'
        THEN '✅ 有工作履历'
        ELSE '❌ 无工作履历'
    END as work_status
FROM candidates
WHERE source = 'github'
  AND talent_labels LIKE '%LLM高质量%'
  AND website_quality_score >= 90
ORDER BY website_quality_score DESC
LIMIT 20;
```
