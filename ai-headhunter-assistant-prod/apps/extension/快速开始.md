# Chrome 插件 - 快速开始

## 📦 安装插件

### 方法 1: Chrome 加载未打包的扩展

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `apps/extension/` 目录
6. 插件图标会出现在浏览器工具栏

### 方法 2: 打包安装

```bash
cd apps/extension
npm run pack

# 会生成 extension.zip
# 拖拽到 chrome://extensions/ 页面安装
```

## 🚀 使用指南

### 基本流程

1. **登录 Boss 直聘**
   - 打开 https://www.zhipin.com
   - 登录你的账号

2. **进入推荐页面**
   - 访问 https://www.zhipin.com/web/chat/recommend
   - 等待候选人列表加载

3. **打开插件控制面板**
   - 插件会自动在页面左下角显示控制面板
   - 如果没有显示，刷新页面

4. **配置处理参数**
   - **后台服务状态**: 确认显示"已连接"（绿色）
   - **处理数量**: 输入要处理的候选人数量（建议先测试 3-5 个）

5. **开始处理**
   - 点击"🚀 开始处理"按钮
   - 观察实时进度和日志

6. **导出结果**
   - 处理完成后，点击"📊 导出结果 (CSV)"
   - 点击"📝 导出原文 (Markdown)"
   - 截图会自动单独下载为 PNG 文件

### 导出文件说明

处理完成后会生成：
- `AI猎头助手-{日期}.csv` - 所有候选人数据表
- `AI猎头助手-{日期}.md` - 详细分析报告
- `{姓名}_简历_{时间戳}.png` - 推荐候选人的简历截图（每人一个）

## ⚙️ 配置后端服务

插件需要连接后端 API 才能进行 AI 评估。

### 后端地址配置

默认后端地址: `http://localhost:8000`

如需修改，编辑 `content-full.js`:
```javascript
const API_CONFIG = {
  baseURL: 'http://localhost:8000',  // 修改为你的后端地址
  timeout: 60000
};
```

### 启动后端

```bash
cd apps/backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## 🐛 故障排查

### 插件没有显示

1. 确认在正确的页面: `https://www.zhipin.com/web/chat/recommend`
2. 刷新页面（F5）
3. 打开浏览器控制台（F12）查看错误日志

### 后台服务未连接

1. 确认后端服务已启动
2. 访问 http://localhost:8000/health 测试连接
3. 检查浏览器控制台的网络请求

### 找不到候选人

1. 等待页面完全加载
2. 手动滚动页面，确保候选人列表已渲染
3. 重新点击"开始处理"

### 截图失败

1. 确认插件权限已授予
2. 检查控制台日志
3. 可能是 Canvas 权限问题，已自动使用截屏 API
