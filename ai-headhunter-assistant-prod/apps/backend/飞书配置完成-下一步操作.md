## 🐛 常见问题

### Q1: Token 获取失败

**错误**: `获取 token 失败`

**解决方案**:
1. 检查 App ID 和 App Secret 是否正确
2. 确认飞书应用已启用
3. 检查网络能否访问 https://open.feishu.cn

### Q2: 上传飞书失败

**错误**: `飞书上传失败`

**可能原因**:
1. Table ID 或 App Token 错误
2. 权限不足（需要添加多维表格读写权限）
3. 表格字段名不匹配

**解决方案**:
1. 检查 `.env` 中的配置是否正确
2. 在飞书开放平台确认应用权限
3. 确保表格字段名与代码中一致

### Q3: 插件无法连接后端

**错误**: `Failed to fetch`

**解决方案**:
1. 确认后端服务已启动（http://localhost:8000）
2. 检查 CORS 配置
3. 查看浏览器控制台的详细错误信息
