# 飞书开放平台注册指南

本指南将帮助你完成飞书开放平台应用的注册和配置。

---

## 📋 前置条件

1. **飞书账号**: 需要有飞书企业账号（个人账号也可以创建企业）
2. **管理员权限**: 需要有企业管理员权限才能创建应用
3. **网络环境**: 需要能正常访问飞书开放平台

---

## 🚀 步骤 1: 创建飞书企业（如已有可跳过）

### 1.1 下载飞书客户端

- 官网: https://www.feishu.cn/
- 下载桌面版或使用网页版

### 1.2 注册并创建企业

1. 使用手机号注册飞书账号
2. 登录后，选择"创建企业"
3. 填写企业信息（名称可随意填写）
4. 完成企业创建

---

## 🔧 步骤 2: 创建飞书开放平台应用

### 2.1 访问飞书开放平台

访问: https://open.feishu.cn/

### 2.2 登录并进入开发者后台

1. 点击右上角"登录"
2. 使用飞书账号登录
3. 进入"开发者后台"

### 2.3 创建企业自建应用

1. 点击"创建企业自建应用"
2. 填写应用信息:
   - **应用名称**: AI 猎头助手
   - **应用描述**: 自动化候选人简历筛选与评估系统
   - **应用图标**: 上传一个图标（可选）

3. 点击"创建"

### 2.4 获取应用凭证

创建成功后，进入应用详情页：

1. 在左侧菜单找到"凭证与基础信息"
2. 记录以下信息:
   - **App ID**: `cli_xxxxxxxxxxxxxxxx`
   - **App Secret**: `xxxxxxxxxxxxxxxxxxxxxxxx`

**⚠️ 重要**: App Secret 非常重要，请妥善保管，不要泄露！

---

## 🔐 步骤 3: 配置应用权限

### 3.1 添加多维表格权限

1. 在应用管理页面，左侧菜单找到"权限管理"
2. 点击"添加权限"
3. 搜索并添加以下权限:
   - ✅ `bitable:app` - 获取多维表格元信息
   - ✅ `bitable:app:readonly` - 查看多维表格
   - ✅ `bitable:app:readwrite` - 编辑多维表格

### 3.2 添加文件上传权限

继续添加以下权限:
   - ✅ `drive:drive` - 访问云空间文件
   - ✅ `drive:drive:readonly` - 查看云空间文件
   - ✅ `drive:drive:readwrite` - 编辑云空间文件

### 3.3 保存权限配置

点击"保存"按钮，完成权限配置。

---

## 📊 步骤 4: 创建多维表格

### 4.1 创建表格

1. 打开飞书客户端
2. 点击左侧"云文档"
3. 点击右上角"新建" → "多维表格"
4. 命名为: **AI 猎头助手 - 候选人库**

### 4.2 设计表格结构

在多维表格中创建以下字段：

| 字段名 | 类型 | 说明 |
|-------|------|------|
| 候选人ID | 文本 | 自动生成 |
| 姓名 | 文本 | 必填 |
| 年龄 | 数字 | - |
| 工作年限 | 数字 | - |
| 学历 | 单选 | 本科/硕士/博士 |
| 当前公司 | 文本 | - |
| 当前职位 | 文本 | - |
| 期望薪资 | 文本 | - |
| 匹配度 | 进度条 | 0-100 |
| 推荐等级 | 单选 | 推荐/一般/不推荐 |
| 核心优势 | 多行文本 | AI 生成 |
| 潜在风险 | 多行文本 | AI 生成 |
| 简历截图 | 附件 | PNG 文件 |
| 数据来源 | URL | Boss 直聘链接 |
| 活跃度 | 文本 | - |
| 到岗状态 | 单选 | 在职/离职 |
| 采集时间 | 日期时间 | 自动 |

### 4.3 获取表格 ID

1. 在浏览器中打开这个多维表格
2. 从 URL 中复制表格 ID

URL 格式: `https://xxx.feishu.cn/base/[TABLE_ID]?...`

例如: `https://xxx.feishu.cn/base/bascnAbCdEfGhIjKlMnOpQr`

这里的 `bascnAbCdEfGhIjKlMnOpQr` 就是你的 **TABLE_ID**

## 🔑 步骤 5: 配置后端环境变量

### 5.1 创建 .env 文件

在 `apps/backend/` 目录下创建 `.env` 文件：

```bash
cd /Users/lillianliao/notion_rag/ai-headhunter-assistant/apps/backend
cp env.example .env
```

### 5.2 填写飞书配置

编辑 `.env` 文件，填入你的飞书信息：

```env
# AI 配置
AI_PROVIDER=qwen
AI_MODEL=qwen-turbo
AI_API_KEY=your-qwen-api-key-here

# 飞书配置
FEISHU_APP_ID=cli_xxxxxxxxxxxxxxxx        # 你的 App ID
FEISHU_APP_SECRET=xxxxxxxxxxxxxxxxxxxxxxxx  # 你的 App Secret
FEISHU_TABLE_ID=bascnAbCdEfGhIjKlMnOpQr    # 你的 Table ID
FEISHU_ENABLED=true                         # 启用飞书集成

# 应用配置
DEBUG=true
API_HOST=0.0.0.0
API_PORT=8000
```

**⚠️ 注意**: `.env` 文件包含敏感信息，不要提交到 Git！

## 🔍 常见问题

### Q1: Token 获取失败

**可能原因**:
1. App ID 或 App Secret 错误
2. 应用未启用
3. 网络连接问题

**解决方案**:
1. 检查 `.env` 文件中的凭证是否正确
2. 在飞书开放平台确认应用状态为"已启用"
3. 检查网络能否访问 `https://open.feishu.cn`

---

### Q2: 权限不足

**错误信息**: `Permission denied`

**解决方案**:
1. 检查是否已添加所有必需的权限
2. 权限修改后需要重新发布应用
3. 确保你的飞书账号有企业管理员权限

---

### Q3: 找不到表格

**错误信息**: `Table not found`

**解决方案**:
1. 检查 TABLE_ID 是否正确
2. 确保表格在你的飞书企业中
3. 确保应用有访问该表格的权限
