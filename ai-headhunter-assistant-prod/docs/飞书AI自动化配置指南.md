# 飞书 AI 自动化配置指南

本指南将帮助你配置飞书多维表格的 AI 自动化功能，实现简历截图的自动解析。

---

## 🎯 实现效果

```
上传简历截图到飞书
    ↓ (自动触发)
飞书 AI 解析图片内容
    ↓
自动提取信息
    ├─ 教育经历
    ├─ 工作经历
    ├─ 技能信息
    └─ 项目经验
    ↓
自动填回表格字段
```

---

## 📋 前置条件

1. ✅ 已完成飞书开放平台应用注册
2. ✅ 已创建多维表格
3. ✅ 飞书企业版（支持自动化功能）
4. ✅ 有多维表格的编辑权限

## 🔧 步骤 1: 准备表格字段

### 1.1 确保表格包含以下字段

| 字段名 | 类型 | 说明 |
|-------|------|------|
| 候选人ID | 文本 | 唯一标识 |
| 姓名 | 文本 | 候选人姓名 |
| 简历截图 | 附件 | **触发字段**（上传图片） |
| AI解析内容 | 多行文本 | **目标字段**（AI 输出） |
| 教育经历 | 多行文本 | AI 提取 |
| 工作经历 | 多行文本 | AI 提取 |
| 技能清单 | 多行文本 | AI 提取 |
| 项目经验 | 多行文本 | AI 提取 |

### 1.2 添加字段（如果还没有）

1. 在多维表格中，点击列标题右侧的 `+`
2. 选择字段类型
3. 命名字段
4. 保存

---

## 🤖 步骤 2: 创建自动化流程

### 2.1 进入自动化设置

1. 打开你的多维表格
2. 点击右上角的 **"自动化"** 按钮
3. 点击 **"创建自动化"**

### 2.2 配置触发条件

**触发器**: 当字段更新时

1. 选择触发器类型: **"当字段更新时"**
2. 选择表格: 当前表格
3. 选择字段: **"简历截图"**
4. 条件: **"字段不为空"**

### 2.3 添加 AI 节点

点击 **"添加步骤"** → 选择 **"AI 生成文本"**

#### 配置 AI 生成文本节点

**节点名称**: AI 解析简历截图

**AI 模型**: 
- 选择 **"GPT-4 Vision"** 或 **"飞书 LLM"**
- （推荐使用 GPT-4 Vision，图片识别效果更好）

**输入内容**:

```
你是一位专业的简历解析助手。请仔细分析这张简历截图，提取以下关键信息：

简历图片：{{记录.简历截图[0].url}}

请按以下格式输出（使用 Markdown 格式）：

## 📚 教育经历
- [学校名称] | [学历] | [专业] | [时间]
（如有多个，每个一行）

## 💼 工作经历
- [公司名称] | [职位] | [时间]
  主要工作内容：xxxxx
（如有多个，每个一行）

## 🛠️ 技能清单
- 编程语言：xxxxx
- 框架/工具：xxxxx
- 其他技能：xxxxx

## 📦 项目经验
- [项目名称]：项目描述
（如有多个，每个一行）

## 💡 其他信息
- 证书/奖项：xxxxx
- 自我评价：xxxxx

请确保：
1. 信息准确，不要编造
2. 格式整齐，使用 Markdown
3. 如果某项信息在简历中没有，标注为"未提供"
```

**输出变量名称**: `ai_parsed_content`

### 2.4 添加更新字段节点

点击 **"添加步骤"** → 选择 **"更新记录"**

#### 配置更新记录节点

**更新的表格**: 当前表格
**更新的记录**: 当前记录

**更新字段**:

1. **AI解析内容** = `{{ai_parsed_content}}`

2. **教育经历** = 使用公式提取:
```
{{ai_parsed_content}}
（提取 ## 📚 教育经历 部分）
```

3. **工作经历** = 使用公式提取:
```
{{ai_parsed_content}}
（提取 ## 💼 工作经历 部分）
```

4. **技能清单** = 使用公式提取:
```
{{ai_parsed_content}}
（提取 ## 🛠️ 技能清单 部分）
```

**简化版本**（推荐）:

如果提取太复杂，可以只更新一个字段：

- **AI解析内容** = `{{ai_parsed_content}}`

其他字段手动复制粘贴。

### 2.5 保存并启用

1. 点击 **"保存"**
2. 给自动化流程命名: **"简历截图自动解析"**
3. 点击 **"启用"**

1. 查看 **"AI解析内容"** 字段

## 🎨 步骤 4: 优化 AI Prompt（可选）

### 4.1 针对特定岗位优化

如果你主要招聘**技术岗位**，可以修改 Prompt：

```
你是一位专业的技术简历解析助手。请分析这张技术人员简历截图：

简历图片：{{记录.简历截图[0].url}}

请重点提取：
1. **技术栈**：编程语言、框架、工具
2. **项目经验**：技术难点、解决方案、成果
3. **开源贡献**：GitHub、技术博客
4. **技术深度**：是否有深度技术积累

按以下格式输出...
```

### 4.2 针对产品岗位优化

如果你主要招聘**产品经理**，可以修改 Prompt：

```
你是一位专业的产品简历解析助手。请分析这张产品经理简历截图：

简历图片：{{记录.简历截图[0].url}}

请重点提取：
1. **产品经验**：负责的产品类型、用户规模
2. **核心能力**：需求分析、产品设计、数据驱动
3. **项目成果**：用户增长、转化率提升等数据
4. **工具能力**：Axure、Figma、SQL 等

按以下格式输出...
```

## 🔍 步骤 5: 进阶配置（可选）

### 5.1 添加条件判断

**场景**: 只对"推荐"等级的候选人执行 AI 解析（节省成本）

1. 在触发器后添加 **"条件判断"** 节点
2. 条件: `记录.推荐等级 = "推荐"`
3. 满足条件才执行 AI 节点

### 5.2 添加通知

**场景**: AI 解析完成后通知你

1. 在最后添加 **"发送通知"** 节点
2. 通知对象: 你自己
3. 通知内容: 
```
候选人 {{记录.姓名}} 的简历已自动解析完成！
匹配度：{{记录.匹配度}}%
推荐等级：{{记录.推荐等级}}
```

### 5.3 添加错误处理

1. 在 AI 节点后添加 **"条件判断"**
2. 条件: `ai_parsed_content 不为空`
3. 如果为空，发送错误通知

## 🐛 常见问题

### Q1: AI 节点没有触发

**可能原因**:
1. 自动化未启用
2. 触发条件不满足
3. 图片字段为空

**解决方案**:
1. 检查自动化状态（应该显示"已启用"）
2. 确认触发条件配置正确
3. 确保上传了图片

---

### Q2: AI 解析结果不准确

**可能原因**:
1. 图片不清晰
2. Prompt 不够明确
3. AI 模型选择不当

**解决方案**:
1. 上传高清截图（建议 > 1MB）
2. 优化 Prompt，增加更多细节要求
3. 尝试切换到 GPT-4 Vision

---

### Q3: 解析速度太慢

**可能原因**:
1. 图片太大
2. AI 模型响应慢
3. 网络问题

**解决方案**:
1. 压缩图片（保持清晰度）
2. 选择飞书 LLM（速度更快）
3. 检查网络连接

---

### Q4: 提取的字段不完整

**可能原因**:
1. 简历格式特殊
2. Prompt 不够具体
3. AI 识别失败

**解决方案**:
1. 优化 Prompt，增加示例
2. 手动补充缺失信息
3. 对特殊格式简历单独处理
