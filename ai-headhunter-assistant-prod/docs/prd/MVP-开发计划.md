# MVP 开发计划 - 2周快速交付

> **目标**：用最快速度实现核心价值，验证完整流程

---

## 🎯 MVP 范围界定

### ✅ 包含功能

1. **Chrome 插件（简化版）**
   - 候选人列表采集
   - 简历下载
   - 调用后台 API
   - 实时进度显示

2. **后台服务（核心功能）**
   - PDF 简历解析
   - AI 结构化提取
   - AI 匹配度评估
   - 返回评估结果

3. **数据导出（手动）**
   - 导出 CSV / Excel
   - 人工导入飞书表格

### ❌ 暂不包含

- ❌ 自动打招呼（风险高，后续添加）
- ❌ 飞书 API 自动入库（手动导入即可）
- ❌ 任务队列（Celery）
- ❌ OCR 支持（先只支持 PDF）
- ❌ 多账号管理
- ❌ 复杂的异常处理

**任务清单**：

- [ ] 将 PDF 转为 Base64
- [ ] 调用后台 API 发送数据
- [ ] 显示后台返回的评估结果

**技术要点**：
```javascript
// 下载简历
async function downloadResume(candidateUrl) {
  // 1. 点击进入详情页
  window.location.href = candidateUrl;
  await sleep(2000);
  
  // 2. 检测简历格式
  const pdfLink = document.querySelector('.resume-download');
  if (pdfLink) {
    // 下载 PDF
    const blob = await fetch(pdfLink.href).then(r => r.blob());
    return await blobToBase64(blob);
  } else {
    // 暂不支持网页简历
    return null;
  }
}

// 上传到后台
async function sendToBackend(candidateInfo, resumeBase64) {
  const response = await fetch('http://localhost:8000/api/candidates/process', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      candidateInfo,
      resumeFile: resumeBase64,
      jdConfig: getJDConfig()  // 从配置获取
    })
  });
  
  return await response.json();
}
```

**Cursor 指令**：
```
"在插件中实现简历下载和上传：
1. 自动点击候选人进入详情页
2. 检测简历格式（优先 PDF）
3. 下载 PDF 并转 Base64
4. 调用后台 API 上传
5. 在控制面板显示后台返回的评估结果
6. 处理完后返回列表页继续下一位"
```

**测试方法**：
