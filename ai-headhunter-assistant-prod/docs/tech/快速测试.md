# ⚡ 快速测试指南 - 5分钟安装

## 🎯 目标

5 分钟内安装并测试插件的基本功能。

---

## 📝 测试步骤

### ✅ Step 1: 打开 Chrome 扩展程序（30秒）

**复制这个链接到 Chrome 地址栏**：
```
chrome://extensions/
```

然后按回车。

---

### ✅ Step 2: 启用开发者模式（10秒）

在扩展程序页面：
1. 找到右上角的**"开发者模式"**开关
2. 打开它（变成蓝色）

---

### ✅ Step 3: 加载插件（1分钟）

1. 点击左上角的**"加载已解压的扩展程序"**按钮

2. 在弹出的文件选择窗口，导航到：
   ```
   /Users/lillianliao/notion_rag/ai-headhunter-assistant/chrome-extension
   ```

3. 点击**"选择"**按钮

4. **验证安装成功**：
   - ✅ 扩展程序列表出现"AI猎头助手"
   - ✅ 状态显示"已启用"
   - ✅ 没有错误提示

---

### ✅ Step 4: 确认后台服务运行（30秒）

**在终端运行**：
```bash
curl http://localhost:8000/health
```

**预期输出**：
```json
{
  "status": "ok",
  "service": "AI Headhunter API",
  "version": "1.0.0-mvp"
}
```

**如果失败**，启动后台：
```bash
cd /Users/lillianliao/notion_rag/ai-headhunter-assistant/backend
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

---

### ✅ Step 5: 打开 Boss 直聘（30秒）

**复制这个链接到 Chrome**：
```
https://www.zhipin.com/web/geek/recommend
```

**或者**手动登录 Boss 直聘，进入"推荐牛人"页面。

---

### ✅ Step 6: 查看控制面板（10秒）

等待 2-3 秒，页面左下角应该出现：

```
┌─────────────────────────┐
│  🤖 AI猎头助手           │
├─────────────────────────┤
│  后台服务: ●已连接       │
│  处理数量: [10]          │
│  [🚀 开始处理]          │
└─────────────────────────┘
```

**如果没有出现**：
1. 刷新页面（F5）
2. 按 F12 打开控制台查看错误
3. 确认 URL 是 `zhipin.com`

---

### ✅ Step 7: 测试处理（3分钟）

1. **打开浏览器控制台**（按 F12）
   - 这样可以看到详细日志

2. **修改处理数量**
   - 在控制面板输入：`3`（先测试 3 个）

3. **点击"🚀 开始处理"**

4. **观察过程**：
   - ✅ 进度条开始移动
   - ✅ 状态文字更新："正在处理: XXX"
   - ✅ 统计数字增加
   - ✅ 控制台有日志输出

5. **等待完成**（约 3 分钟）

---

### ✅ Step 8: 导出结果（30秒）

1. **点击"📊 导出结果"**按钮

2. **下载 CSV 文件**
   - 文件名：`AI猎头助手-候选人评估-2025-11-13.csv`

3. **打开 CSV 文件**
   - 用 Excel 或 WPS 打开
   - 检查数据是否完整

## 🐛 快速故障排查

### ❌ 插件加载失败

**检查**：
```bash
# 验证文件是否存在
ls /Users/lillianliao/notion_rag/ai-headhunter-assistant/chrome-extension/manifest.json
```

---

### ❌ 控制面板不显示

**解决**：
1. 确认在 Boss 直聘网站（URL 包含 zhipin.com）
2. 刷新页面（F5）
3. 按 F12 查看控制台错误

---

### ❌ 后台未连接

**检查后台**：
```bash
# 测试后台
curl http://localhost:8000/health

# 如果失败，重启后台
cd /Users/lillianliao/notion_rag/ai-headhunter-assistant/backend
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

---

### ❌ 处理时报错

**查看**：
1. 浏览器控制台的错误信息
2. 后台终端的日志
3. 通义千问 API 额度是否充足
