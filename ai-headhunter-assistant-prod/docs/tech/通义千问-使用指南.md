# 🚀 通义千问 API 使用指南

## 1. 申请 API Key

### Step 1: 注册阿里云账号

1. 访问 https://www.aliyun.com
2. 使用手机号注册（支持支付宝登录）
3. 完成实名认证

---

### Step 2: 开通通义千问服务

1. 访问 **百炼平台**：https://bailian.console.aliyun.com/
2. 点击「开通服务」
3. 同意服务协议

---

### Step 3: 获取 API Key

1. 在百炼平台，点击「API-KEY」
2. 点击「创建新的API-KEY」
3. 复制 API Key（格式：`sk-...`）

**⚠️ 重要**：API Key 只显示一次，务必保存好！

---

## 3. 使用 Demo

### 基础用法（通义千问）

```bash
python demo_ai_resume_parser.py \
  --resume <简历路径> \
  --api-key <通义千问API Key> \
  --provider qwen \
  --model qwen-turbo
```

### 实际示例

```bash

  --resume ~/Downloads/resume.pdf \
  --api-key sk-xxxxxxxxxxxxx \
  --provider qwen \
  --model qwen-turbo
```

## 4. 模型选择

| 模型 | 价格 | 速度 | 适用场景 | 推荐 |
|------|------|------|---------|------|
| **qwen-turbo** | 最便宜 | 最快 | 日常使用 | ⭐⭐⭐⭐⭐ |
| **qwen-plus** | 中等 | 中等 | 复杂简历 | ⭐⭐⭐ |

## 5. 命令行参数说明

```bash
python demo_ai_resume_parser.py \
  --resume <PDF路径>           # 必填：简历文件路径
  --api-key <API Key>          # 必填：通义千问 API Key
  --provider qwen              # 必填：使用通义千问
  --model qwen-turbo           # 可选：模型名称（默认 qwen-turbo）
```

### 切换到 OpenAI（如果需要对比）

## 6. 常见问题

### Q1: API Key 无效

**错误信息**：`Invalid API key`

**解决方案**：
1. 检查 API Key 是否正确复制（完整的 `sk-...`）
2. 确认已开通百炼服务
3. 尝试重新创建 API Key

---

### Q2: 余额不足

**错误信息**：`Insufficient balance`

**解决方案**：
1. 登录阿里云控制台
2. 进入「账户管理」
3. 充值（最低 ¥10）

---

### Q3: 解析结果不准确

**原因**：
- Prompt 需要优化
- 简历格式特殊
- 模型能力限制

**解决方案**：
1. 尝试升级到 `qwen-plus` 或 `qwen-max`
2. 调整 Prompt（编辑 `demo_ai_resume_parser.py`）
3. 提供更清晰的 JD 配置

---

### Q4: 速度太慢

**原因**：
- 简历太长
- 网络延迟
- 模型处理时间

**优化方案**：
1. 限制简历长度（前 8000 字符）
2. 使用 `qwen-turbo`（最快）
3. 升级网络环境

---

### Q5: 想要更长的上下文

**场景**：简历超过 10 页

**方案 1**：使用 `qwen-long`
```bash
--model qwen-long  # 支持 30K tokens
```

**方案 2**：切换到 Kimi
```bash
--provider kimi \
--model moonshot-v1-128k  # 支持 128K tokens
```
