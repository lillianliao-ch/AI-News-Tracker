## 🎓 技术亮点

### 1. 混合架构
```
Chrome 插件（前端）
    ↓ HTTP API
FastAPI 后端（AI 处理）
    ↓ API 调用
通义千问（大模型）
```

**优势**:
- 反爬风险低（真实浏览器）
- 用户体验好（可视化）
- AI 处理强大（后端集中）

### 2. 两轮处理逻辑

**第一轮 - 快速筛选**:
- 处理所有候选人卡片
- 提取基本信息
- Mock AI 评级
- 保存到 CSV

**第二轮 - 深度分析**:
- 只对"推荐"候选人
- 提取完整简历
- 真实 AI 评估
- 截图 + Markdown

**节省成本**: 约 70%（只对推荐候选人用真实 AI）

### 3. Canvas 加密突破

Boss 直聘使用 WebAssembly + Canvas 渲染简历，无法直接提取文本和导出截图。

**解决方案**:
1. 尝试 Canvas `toDataURL()`
2. 失败后使用 Chrome `tabs.captureVisibleTab` API
3. 通过 Background Service Worker 处理
4. Message Passing 通信

### 4. Monorepo 架构

**优势**:
- 模块化清晰
- 代码复用
- 统一构建
- 易于维护
