## 🧪 测试步骤

### 测试 1: 基础功能测试（5-10 个候选人）

#### Step 1: 启动处理
1. 打开 Boss 直聘推荐页面
2. 等待插件控制面板出现
3. 点击"开始处理"按钮
4. 设置处理数量: 5-10 个

#### Step 2: 观察第一轮处理
```
✅ 应该看到:
- 候选人信息被提取
- 显示"推荐:X 一般:X 不推荐:X"
- 进度条前进到 50%
```

#### Step 3: 观察第二轮处理（关键！）
```
对于每个"推荐"候选人，应该看到:

1. 📄 "提取简历: 张先生 (1/3)"
2. 📸 "简历截图已保存"
3. ⭐ "开始自动收藏和转发 张先生..."
4. ⭐ "张先生 收藏成功" 或 "张先生 收藏(已收藏)"
5. 📧 "张先生 已转发到 liao412@gmail.com"
```

#### Step 4: 验证结果

**验证收藏**:
1. 在 Boss 直聘页面，点击顶部"收藏"
2. 查看是否有新收藏的候选人
3. 确认候选人姓名匹配

**验证转发**:
1. 打开邮箱: liao412@gmail.com
2. 查看是否收到来自 Boss 直聘的转发邮件
3. 确认邮件数量与推荐候选人数量一致

## 🐛 常见问题排查

### 问题 1: 收藏按钮未找到

**症状**:
```
❌ 未找到收藏按钮
```

**排查步骤**:
1. 打开浏览器控制台（F12）
2. 在 Elements 标签页搜索"收藏"
3. 查看收藏按钮的 class 名称
4. 对比代码中的选择器

**解决方案**:
```javascript
// 如果按钮 class 是 'btn-star'，添加到选择器列表:
const selectors = [
  'button.btn-collect',
  '.collect-btn',
  'button.btn-star',  // 新增
  // ...
];
```

---

### 问题 2: 转发邮件未收到

**可能原因**:
1. Boss 直聘限流（发送过于频繁）
2. 邮箱地址填写失败
3. 发送按钮未点击成功

**排查步骤**:
1. 查看控制台日志，确认"已转发"消息
2. 在 Boss 直聘查看"已发送"记录
3. 检查邮箱垃圾邮件文件夹

**解决方案**:
- 增加转发之间的延迟时间
- 手动测试转发流程，确认页面元素

---

### 问题 3: 插件卡住不动

**症状**:
- 进度条停止
- 日志不再更新
- 页面无响应

**排查步骤**:
1. 打开浏览器控制台（F12）
2. 查看 Console 标签页的错误信息
3. 查看 Network 标签页的网络请求

**解决方案**:
1. 刷新页面
2. 重新加载插件
3. 检查后端是否运行
4. 查看错误日志，定位问题

---

### 问题 4: 收藏/转发速度太慢

**症状**:
- 每个候选人处理时间过长
- 整体流程耗时太久

**优化方案**:
```javascript
// 减少延迟时间（需谨慎，避免触发反爬）
await Utils.sleep(500);  // 从 1000 改为 500
```
