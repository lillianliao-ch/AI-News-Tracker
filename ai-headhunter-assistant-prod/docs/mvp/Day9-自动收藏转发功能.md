# Day 9: 自动收藏和转发功能

> **日期**: 2025-11-15  
> **状态**: ✅ 已完成  
> **功能**: 对推荐候选人自动执行收藏和邮件转发

---

## 📋 功能概述

为 MVP 新增两个自动化操作功能，针对**评分高（"推荐"等级）**的候选人：

1. ✅ **自动收藏**: 点击 Boss 页面的"收藏"按钮
2. ✅ **自动转发**: 点击"转发牛人"，发送到 `liao412@gmail.com`

---

## 🎯 功能特点

### 1. 智能触发
- 只对"推荐"等级的候选人执行
- 在第二轮详细简历提取后自动触发
- 与现有流程无缝集成

### 2. 容错机制
- 多选择器兼容（适配不同页面版本）
- 已收藏检测（避免重复收藏）
- 失败不中断流程（记录警告但继续）

### 3. 用户友好
- 详细的日志输出
- 成功/失败状态反馈
- 自动延迟避免频繁操作

## 🔍 选择器适配

### 收藏按钮选择器
```javascript
[
  'button.btn-collect',      // 标准收藏按钮
  '.collect-btn',            // 通用收藏按钮
  'button[class*="collect"]', // 包含 collect 的按钮
  'button[class*="favorite"]', // 包含 favorite 的按钮
  '.icon-collect',           // 收藏图标
  'button:has(.icon-collect)' // 包含收藏图标的按钮
]
```

### 转发按钮选择器
```javascript
[
  'button.btn-forward',       // 标准转发按钮
  '.forward-btn',             // 通用转发按钮
  'button[class*="forward"]',  // 包含 forward 的按钮
  'button[class*="share"]',    // 包含 share 的按钮
  'a[href*="forward"]'         // 转发链接
]
```

### 邮箱输入框选择器
```javascript
[
  'input[type="email"]',           // 标准邮箱输入
  'input[placeholder*="邮箱"]',     // 包含"邮箱"的输入框
  'input[placeholder*="收件人"]',   // 包含"收件人"的输入框
  '.email-input input',            // 邮箱输入容器
  'input[class*="email"]'          // 包含 email 的输入框
]
```

## ⚠️ 注意事项

### 1. Boss 直聘页面变化
- Boss 直聘可能更新页面结构
- 选择器可能失效
- 建议定期检查和更新选择器

### 2. 频率限制
- 避免过于频繁的操作
- 已添加 1 秒延迟（收藏和转发之间）
- 遵循 Boss 直聘的使用规则

### 3. 邮箱地址
- 当前硬编码为 `liao412@gmail.com`
- 如需修改，更新代码中的 email 参数
- 未来可考虑添加配置界面

### 4. 错误处理
- 收藏失败不影响转发
- 转发失败不影响后续候选人
- 所有错误都会记录日志
