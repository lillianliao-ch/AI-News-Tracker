## 🐛 常见问题快速解决

### ❌ 控制面板不显示

```bash
# 1. 检查是否在正确页面
当前 URL 应该包含: zhipin.com

# 2. 刷新页面
按 F5 刷新

# 3. 查看控制台错误
按 F12 查看 Console 标签
```

---

### ❌ 后台服务未连接

```bash
# 1. 检查后台是否启动
curl http://localhost:8000/health

# 2. 如果没有响应，重新启动后台
cd backend
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000

# 3. 确认服务正常后，刷新 Boss 页面
```

---

### ❌ 处理时报错

```bash
# 1. 查看控制台错误信息
按 F12 → Console 标签

# 2. 常见错误：
- "AI API 失败" → 检查通义千问 API Key 和余额
- "网络错误" → 检查后台服务是否运行
- "超时" → AI 处理较慢，等待更长时间

# 3. 重试
点击"停止"，然后重新"开始处理"
```
