# Chrome 插件 - AI猎头助手

## 📋 功能说明

自动采集 Boss 直聘候选人信息，调用后台 AI 服务进行智能评估。

**核心功能**：
- ✅ 自动读取候选人列表
- ✅ 自动采集候选人信息
- ✅ 调用后台 API 进行 AI 评估
- ✅ 实时显示处理进度
- ✅ 导出评估结果（CSV）

---

## 🚀 安装步骤

### 1. 确保后台服务运行

```bash
cd backend
python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

服务启动后访问 http://localhost:8000/health 确认正常。

---

### 2. 准备插件图标（可选）

在 `icons/` 目录下放置以下尺寸的图标：
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

如果没有图标，可以暂时跳过（插件仍可正常工作）。

---

### 3. 加载插件到 Chrome

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 打开右上角的**"开发者模式"**
4. 点击**"加载已解压的扩展程序"**
5. 选择 `chrome-extension` 目录
6. 点击"选择"

![安装插件](https://via.placeholder.com/600x300?text=Load+Extension)

插件安装成功后，会在 Chrome 工具栏显示图标。

---

## 📖 使用指南

### Step 1: 打开 Boss 直聘

访问：https://www.zhipin.com/web/geek/recommend

或进入任何候选人列表页面。

---

### Step 2: 控制面板自动出现

插件会在页面左下角显示控制面板：

```
┌─────────────────────────┐
│  🤖 AI猎头助手           │
├─────────────────────────┤
│  后台服务: ●已连接       │
│  处理数量: [10]          │
│  [🚀 开始处理]          │
│  进度: 0 / 0 (0%)        │
│  已处理: 0              │
│  ✅ 推荐: 0             │
│  [📊 导出结果]          │
└─────────────────────────┘
```

---

### Step 3: 设置处理数量

在"处理数量"输入框中设置要处理的候选人数量（建议 10-20）。

---

### Step 4: 开始处理

点击**"🚀 开始处理"**按钮，插件会自动：

1. ✅ 读取候选人列表
2. ✅ 提取候选人信息
3. ✅ 调用后台 API
4. ✅ AI 评估匹配度
5. ✅ 更新进度和统计

---

### Step 5: 查看结果

处理完成后：
- 查看面板统计（推荐/一般/不推荐）
- 点击**"📊 导出结果"**按钮
- 自动下载 CSV 文件

---

### Step 6: 导入飞书

将下载的 CSV 文件导入飞书多维表格，完成候选人入库。

---

## 🎮 功能说明

### 控制按钮

| 按钮 | 功能 |
|------|------|
| 🚀 开始处理 | 开始自动处理候选人 |
| ⏸ 暂停 | 暂停当前任务 |
| ⏹ 停止 | 停止任务 |
| 📊 导出结果 | 导出 CSV 文件 |
| − 最小化 | 最小化控制面板 |

---

### 统计信息

| 项目 | 说明 |
|------|------|
| 已处理 | 总共处理的候选人数量 |
| ✅ 推荐 | 匹配度 ≥ 75% 的候选人 |
| ⚠️ 一般 | 匹配度 60-74% 的候选人 |
| ❌ 不推荐 | 匹配度 < 60% 的候选人 |
| 失败 | 处理失败的候选人 |

---

## 📊 导出格式

CSV 文件包含以下字段：

| 字段 | 说明 |
|------|------|
| 姓名 | 候选人姓名 |
| 当前公司 | 当前公司名称 |
| 当前职位 | 当前职位 |
| 期望薪资 | 期望薪资范围 |
| 匹配度 | AI 评估的综合匹配度（0-100） |
| 推荐等级 | 推荐 / 一般 / 不推荐 |
| 核心优势 | AI 分析的核心优势 |
| 潜在风险 | AI 分析的潜在风险 |
| 来源链接 | Boss 直聘原始链接 |

## 🛠️ 配置说明

### 修改后台 API 地址

编辑 `lib/config.js`：

```javascript
const CONFIG = {
  API_BASE_URL: 'http://localhost:8000',  // 修改为你的后台地址
  // ...
};
```

---

### 修改 JD 配置

编辑 `lib/config.js` 中的 `DEFAULT_JD`：

```javascript
DEFAULT_JD: {
  position: '你的职位名称',
  location: ['北京', '深圳'],
  salary_range: '30-60K',
  work_years: '3-5年',
  education: '本科及以上',
  required_skills: [
    '技能1',
    '技能2'
  ],
  optional_skills: [
    '加分技能1'
  ]
}
```

---

### 修改处理延迟

编辑 `lib/config.js` 中的 `PROCESSING`：

```javascript
PROCESSING: {
  delay_min: 2000,  // 最小延迟（毫秒）
  delay_max: 5000,  // 最大延迟（毫秒）
  retry_times: 3,   // 重试次数
  timeout: 60000    // 超时时间
}
```

## 🐛 常见问题

### Q1: 控制面板没有出现

**解决**：
1. 确认在 Boss 直聘网站（zhipin.com）
2. 刷新页面（F5）
3. 检查浏览器控制台是否有错误

---

### Q2: 后台服务未连接

**解决**：
1. 确认后台服务是否启动
2. 访问 http://localhost:8000/health 测试
3. 检查 `lib/config.js` 中的 API 地址

---

### Q3: 处理失败

**可能原因**：
- 后台服务异常
- AI API 额度不足
- 网络连接问题
- Boss 页面结构变化

**解决**：
1. 查看浏览器控制台错误信息
2. 查看后台服务日志
3. 重试任务

---

### Q4: 导出的 CSV 乱码

**解决**：
- 使用 Excel 打开 CSV 文件
- 或使用 WPS Office
- CSV 文件已使用 UTF-8 BOM 编码
