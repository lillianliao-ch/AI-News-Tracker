- [ ] 后台服务已启动（http://localhost:8000）

---

## 🔧 安装步骤

### Step 1: 打开 Chrome 扩展程序页面

在 Chrome 地址栏输入：
```
chrome://extensions/
```

或通过菜单：**更多工具 → 扩展程序**

---

### Step 2: 启用开发者模式

在扩展程序页面右上角，打开**"开发者模式"**开关。

![开发者模式](https://via.placeholder.com/600x100?text=Enable+Developer+Mode)

---

### Step 3: 加载插件

1. 点击左上角的**"加载已解压的扩展程序"**
2. 选择插件目录：
   ```
   /Users/lillianliao/notion_rag/ai-headhunter-assistant/chrome-extension
   ```
3. 点击**"选择"**

---

### Step 4: 验证安装

安装成功后，你会看到：

- ✅ 扩展程序列表中出现"AI猎头助手"
- ✅ 状态显示"已启用"
- ✅ Chrome 工具栏出现插件图标（如果有图标）

**如果出现错误**：
- 检查 `manifest.json` 是否正确
- 查看错误提示信息
- 重新加载插件

---

## 🧪 测试步骤

### 测试 1: 后台服务连接

**目标**：验证插件能连接后台

1. 点击插件图标，打开弹窗页面
2. 查看"后台服务"状态：
   - ✅ "●已连接"（绿色）→ 正常
   - ❌ "●未连接"（红色）→ 检查后台服务

---

### 测试 2: 控制面板显示

**目标**：验证控制面板能正常显示

1. 打开 Boss 直聘：https://www.zhipin.com/web/geek/recommend
2. 等待 2-3 秒
3. 检查页面左下角是否出现控制面板

**如果没有出现**：
- 按 F12 打开控制台，查看错误信息
- 刷新页面（F5）
- 检查是否在正确的 Boss 页面

---

### 测试 3: 候选人列表读取

**目标**：验证能读取候选人列表

1. 确保 Boss 页面有候选人列表
2. 打开浏览器控制台（F12）
3. 在控制台输入：
   ```javascript
   document.querySelectorAll('.recommend-job-list .job-card-wrapper').length
   ```
4. 如果返回数字 > 0，说明能读取列表

---

### 测试 4: 完整流程测试（重要）

**目标**：测试完整的采集→评估流程

**步骤**：

1. **设置处理数量**：输入 `3`（先测试 3 个）

2. **点击"开始处理"**

3. **观察过程**：
   - 进度条更新
   - 状态文字变化
   - 统计数字增加
   - 控制台日志输出

4. **等待完成**（约 3-5 分钟）

5. **查看结果**：
   - 统计数字是否正确
   - 点击"导出结果"
   - 下载的 CSV 文件是否正常

---

### 测试 5: 导出 CSV

**目标**：验证数据导出功能

1. 处理完成后，点击"📊 导出结果"
2. 浏览器会自动下载 CSV 文件
3. 用 Excel 或 WPS 打开 CSV 文件
4. 检查数据是否完整：
   - 姓名、公司、职位
   - 匹配度、推荐等级
   - 核心优势、潜在风险

## 🐛 故障排查

### 问题 1: 插件加载失败

**错误信息**：`Manifest file is missing or unreadable`

**解决**：
1. 检查 `manifest.json` 是否存在
2. 检查 JSON 格式是否正确
3. 重新加载插件

---

### 问题 2: 后台服务未连接

**错误信息**：`后台服务: ●未连接`

**解决**：
1. 确认后台服务是否启动：
   ```bash
   curl http://localhost:8000/health
   ```
2. 检查防火墙设置
3. 检查 CORS 配置

---

### 问题 3: 控制面板不显示

**可能原因**：
- 不在 Boss 直聘页面
- JavaScript 加载失败
- Boss 页面结构变化

**解决**：
1. 确认 URL 包含 `zhipin.com`
2. 打开控制台查看错误
3. 刷新页面

---

### 问题 4: 处理卡住不动

**可能原因**：
- 后台 API 超时
- AI 服务异常
- 网络延迟

**解决**：
1. 点击"停止"按钮
2. 查看后台日志
3. 检查 AI API 额度
4. 重试任务

---
