# 插件图标

## 📝 说明

请在此目录下放置插件图标（PNG 格式）：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

---

## 🎨 如何创建图标？

### 方式 1：在线工具

使用在线图标生成工具：
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

上传一张图片，自动生成各种尺寸的图标。

---

### 方式 2：设计软件

使用 Figma / Sketch / Photoshop 设计图标。

**推荐设计**：
- 主色调：渐变紫色（#667eea → #764ba2）
- 图标内容：🤖 机器人 或 AI 字母
- 风格：简约、现代

---

### 方式 3：临时方案（开发阶段）

如果暂时没有图标，可以使用纯色占位：
- 创建 3 个纯色 PNG 文件（紫色背景 + 白色文字 "AI"）

---

## 💡 提示

没有图标不影响插件功能，只是工具栏会显示默认图标。

开发阶段可以先跳过图标，等功能完善后再补充。
