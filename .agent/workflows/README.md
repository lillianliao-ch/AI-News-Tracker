## 📖 推荐阅读顺序

### 新手入门
1. 主文档 → 路线图（了解 Phase 1-7）
2. 主文档 → 前提准备（配置 Token）
3. 主文档 → Runbook 1（尝试执行一次增量处理）

### 执行任务时
1. 主文档 → 对应的 Runbook
2. 参考文档 → 常用命令速查（如需查找命令）

### 遇到问题时
1. 参考文档 → 已知问题与解决方案
2. 参考文档 → 执行记录（查看类似问题）

### 开发脚本时
1. 参考文档 → 分级标准（确保符合标准）
2. 参考文档 → 字段映射（确保字段正确）
3. 参考文档 → 评分公式（确保评分逻辑正确）
