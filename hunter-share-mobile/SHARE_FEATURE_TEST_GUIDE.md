# 微信分享功能测试指南

## 🎯 测试目标

验证猎头协作平台的微信分享功能是否正常工作，包括：
- Open Graph元标签是否正确
- 分享按钮是否显示和工作
- 复制链接功能
- 二维码生成功能
- 微信分享预览

---

npm install -g localtunnel

## 🐛 常见问题排查

### 问题1: 分享按钮不显示

**排查步骤**:
```bash
# 1. 检查组件是否正确导入
grep -r "ShareButton" frontend/src/app/page.tsx

# 2. 检查npm包是否安装
cd frontend && npm list qrcode.react

# 3. 查看浏览器控制台错误
# 打开Chrome DevTools → Console
```

### 问题2: 二维码不显示

**可能原因**:
- qrcode.react包未正确安装
- 组件导入错误
- SVG渲染问题

**解决方案**:
```bash
cd frontend
npm install qrcode.react --save
npm run dev
```

### 问题3: 复制功能不工作

**可能原因**:
- 浏览器不支持 Clipboard API
- 页面不是HTTPS（生产环境）

**解决方案**:
- 代码中已有降级方案（document.execCommand）
- 在 localhost 环境应该正常工作

### 问题4: 微信不显示预览卡片

**可能原因**:
- 本地环境（localhost）微信无法访问
- Open Graph 标签未正确设置
- 微信缓存了旧的预览

**解决方案**:
1. 使用 ngrok 等工具提供公网访问
2. 检查 Open Graph 标签
3. 清除微信缓存（退出重新登录）
